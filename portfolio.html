Aplikasi Laundry

Aplikasi Laundry sederhana Berbasis Website

Aplikasi ini dibuat menggunakan Bahasa PHP dengan Codeigniter 3 dan Bootstrap 5.Bersifat Open Source,cara pengistalan bisa dilihat di repositori github saya

Kegunaan:

  1. Mempermudah Admin,Kasir,Dan Owner dalam mengelola data Laundry
  2. Mempermudah pelanggan dalam membuat pesanan

Fitur:

  1. Antarmuka Petugas
  2. Antarmuka Pelanggan
  3. Login,logout,register Multi User
  4. Session Hak Akses
  5. Dashboard Petugas

Admin:

  1. CRUD Petugas
  2. CRUD Pelanggan
  3. CRUD Data Paket
  4. CRUD Data Pesanan
  5. Generate Laporan

Kasir:

  1. Transaksi Pesanan
  2. Lihat Data Pelanggan
  3. Lihat Data Paket
  4. Lihat Data Pesanan

Owner:

  1. Lihat Data Pesanan
  2. Lihat Data Pelanggan
  3. Lihat Data Paket
  4. Lihat Data Pesanan
  5. Generate Laporan

Pelanggan:

  1. Lihat Data Pesanan
  2. Lihat Data Paket
  3. Memesan Paket
  4. login,Logout,register
